# Automatic Well Tie
## Instalação
Para instalar os pacotes necessários para rodar a ferramenta execute o seguinte comando:

``` shell
pip install -r requirements.txt
```

## Execução
### Linux
Para executar um dos tipos de amarração sísmica disponíveis, execute:

``` shell
make run file=nome_arquivo
```

Para treinar um dos tipos de amarração sísmica disponíveis, execute:

``` shell
make train file=nome_arquivo
```


### Windows
Para executar um dos tipos de amarração sísmica disponíveis, execute:

``` powershell
make.bat run nome_arquivo.py
```

Para treinar um dos tipos de amarração sísmica disponíveis, execute:

``` powershell
make.bat train nome_arquivo.py
```
